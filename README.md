Developed a lightweight and intuitive desktop utility to perform real-time temperature conversions between Celsius and Fahrenheit. This project was built using C# and .NET Windows Forms, focusing on event-driven programming and mathematical data processing.
Key Features:
Bi-directional Conversion: Supports seamless conversion from Celsius to Fahrenheit and vice versa based on user selection.
Event-Driven UI: Implemented responsive UI interactions where the application processes inputs and updates results dynamically upon the "Convert" action.
Input Handling: Integrated robust logic to manage user input values and perform precise temperature calculations using standard formulas.
Tech Stack: C#, .NET Framework, Windows Forms (WinForms).

How it works: "The user enters a value, selects a conversion type (Celsius to Fahrenheit or Fahrenheit to Celsius) via radio buttons, and clicks the Convert button to view the result."
Celsius to Fahrenheit: (C×9/5)+32
Fahrenheit to Celsius: (F−32)×5/9
